# Decisión visual inicial

## Concepto elegido

El sistema SIGO-PRO FoodOps tendrá una estética moderna inspirada en una landing page tecnológica/gastronómica.

## Elementos principales

- Video de fondo en pantalla de inicio.
- Header tipo glassmorphism.
- Hero inferior izquierdo.
- Tarjetas visuales para módulos.
- Paleta gastronómica: rojo vino, negro carbón, crema y dorado suave.
- Páginas internas limpias, funcionales y consistentes.

## Justificación

Esta decisión busca mejorar la experiencia de usuario, elevar la presentación del sistema y reforzar el rol integrador mediante una interfaz coherente, profesional y orientada a la defensa final del proyecto.

## Restricción

El diseño se implementará en Streamlit, no en React. Por ello se usará CSS personalizado, HTML embebido y componentes reutilizables desde `core/ui.py`.
